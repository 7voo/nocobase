# @ak/plugin-auth-wecom

企业微信认证插件，为 NocoBase 提供企业微信登录功能。

## 功能特性

### 双模式登录支持

1. **PC 扫码登录**
   - 用户在电脑浏览器上访问 NocoBase
   - 点击"使用企业微信登录"按钮
   - 显示二维码
   - 使用企业微信手机客户端扫码确认
   - 自动登录

2. **企业微信内一键登录** ✨ 新功能
   - 用户在企业微信客户端内打开 NocoBase 应用
   - 系统自动检测企业微信环境
   - 点击"使用企业微信登录（一键登录）"按钮
   - 直接跳转到企业微信授权页面
   - 确认后自动登录，无需扫码

### 自动环境检测

插件会自动检测用户的访问环境：
- **企业微信内**：显示一键登录按钮，点击后直接使用 OAuth2.0 授权
- **PC 浏览器**：显示二维码扫码登录

### 其他特性

- 自动用户注册
- 用户信息同步
- 默认角色分配
- 安全的 OAuth 2.0 流程
- CSRF 保护
- 国际化支持（中文/英文）

## 安装

1. 上传插件包到 NocoBase
2. 在插件管理页面启用插件
3. 配置企业微信应用信息

## 配置

### 1. 企业微信应用配置

在企业微信管理后台：

1. 创建自建应用
2. 获取以下信息：
   - 企业 ID (Corp ID)
   - 应用 ID (Agent ID)
   - 应用密钥 (Secret)
3. 配置可信域名（重要！）
   - 将 NocoBase 的域名添加到应用的"可信域名"中
   - 如果使用非标准端口（如 7737），需要包含端口号
   - 例如：`example.com:7737`

### 2. NocoBase 插件配置

在 NocoBase 认证设置页面：

1. 创建新的认证器
2. 选择"企业微信"类型
3. 填写配置信息：
   - **企业 ID**: 从企业微信获取
   - **应用 ID**: 从企业微信获取
   - **应用密钥**: 从企业微信获取
   - **回调地址**: 自动生成，复制到企业微信应用设置
   - **公网端口**: 如果使用非标准端口，填写此项（可选）
   - **允许自动注册**: 是否为新用户自动创建账号
   - **默认角色**: 新用户的默认角色（可选）

## 使用方法

### PC 浏览器登录

1. 访问 NocoBase 登录页面
2. 点击"使用企业微信登录"按钮
3. 弹出二维码对话框
4. 使用企业微信扫描二维码
5. 在手机上确认登录
6. 自动跳转到 NocoBase 主页

### 企业微信内登录

1. 在企业微信客户端内打开 NocoBase 应用
2. 系统自动检测到企业微信环境
3. 点击"使用企业微信登录（一键登录）"按钮
4. 跳转到企业微信授权页面
5. 确认授权
6. 自动登录并跳转到 NocoBase 主页

## 技术实现

### 登录流程

#### PC 扫码登录
- 使用企业微信 PC 端扫码登录接口
- 端点：`https://open.work.weixin.qq.com/wwopen/sso/qrConnect`

#### 企业微信内 OAuth2.0 登录
- 使用企业微信 OAuth2.0 授权接口
- 端点：`https://open.weixin.qq.com/connect/oauth2/authorize`
- Scope: `snsapi_base`（获取基础用户信息）

### 环境检测

通过检测 User-Agent 中的 `wxwork` 标识来判断是否在企业微信环境中。

### 安全特性

- OAuth 2.0 标准流程
- State 参数 CSRF 保护
- HTTPS 通信
- 敏感信息加密存储
- 授权码单次使用，5 分钟过期

## 故障排除

### 常见问题

1. **二维码无法显示**
   - 检查网络连接
   - 确认企业 ID 和应用 ID 配置正确

2. **扫码后提示"redirect_uri 参数错误"**
   - 检查企业微信应用的可信域名配置
   - 确保域名完全匹配（包括端口号）

3. **企业微信内无法一键登录**
   - 确认在企业微信客户端内打开
   - 检查可信域名配置
   - 查看浏览器控制台错误信息

4. **登录后提示"用户不存在"**
   - 启用"允许自动注册"选项
   - 或手动创建用户账号
